# Installation
```bash
npm: npx rlz@latest add avatar
pnpm: pnpm dlx rlz@latest add avatar
yarn: yarn rlz@latest add avatar
bun: bunx --bun rlz@latest add avatar
```

# Usage
```tsx
import { Avatar } from "@ui/components/Avatar"
```

```tsx
<Avatar
  src="/c.gif"
  alt="Avatar"
  fallback="CN"
  size="lg"
/>
```
## or use it as is for better personalisation
```tsx
<Avatar.Root>
  <Avatar.Image src="/c.gif" alt="Avatar" />
  <Avatar.Fallback>CN</Avatar.Fallback>
</Avatar.Root>
```

## Avatar (Main)

An image element with a fallback for representing the user.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | The image source URL |
| `alt` | `string` | - | Alternative text for the image |
| `fallback` | `string` | - | Fallback content when image fails to load (text or image URL) |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | The size of the avatar |
| `className` | `string` | - | Additional CSS classes to apply to the avatar |
| `loading` | `"eager" \| "lazy"` | `"lazy"` | Loading strategy for the image |

## Avatar Root

The root container for the avatar component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The content to render inside the avatar root |
| `className` | `string` | Predefined styles | Additional CSS classes to apply to the root element |

## Avatar Image

The image element of the avatar.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | The image source URL |
| `alt` | `string` | - | Alternative text for the image |
| `className` | `string` | Predefined styles | Additional CSS classes to apply to the image element |

## Avatar Fallback

The fallback content when the image fails to load.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The fallback content to render (usually initials) |
| `className` | `string` | Predefined styles | Additional CSS classes to apply to the fallback element |
