# Installation
```bash
npm: npx rlz@latest add scroll-area
pnpm: pnpm dlx rlz@latest add scroll-area
yarn: yarn rlz@latest add scroll-area
bun: bunx --bun rlz@latest add scroll-area
```

# Usage
```tsx
import { ScrollArea } from "@ui/components/scroll-area";
```

```tsx
<ScrollArea className="h-[200px] w-[350px] rounded-md border p-4">
  Jokester began sneaking into the castle in the middle of the night and leaving
  jokes all over the place: under the king's pillow, in his soup, even in the
  royal toilet. The king was furious, but he couldn't seem to stop Jokester. And
  then, one day, the people of the kingdom discovered that the jokes left by
  Jokester were so funny that they couldn't help but laugh. And once they
  started laughing, they couldn't stop.
</ScrollArea>
```

```tsx
<ScrollArea.Root className="h-72 w-48 border rounded-md">
  <ScrollArea.Viewport>
    {/* Content here */}
  </ScrollArea.Viewport>
  <ScrollArea.ScrollBar orientation="vertical" />
  <ScrollArea.ScrollBar orientation="horizontal" />
</ScrollArea.Root>
```

## ScrollArea (Main)

| Prop        | Type                                         | Description                | Default |
|-------------|----------------------------------------------|----------------------------|---------|
| children    | `ReactNode`                                  | Scrollable content.        | -       |
| className   | `string`                                     | Additional CSS classes.    | -       |

## ScrollArea.Root

| Prop        | Type                                         | Description                | Default |
|-------------|----------------------------------------------|----------------------------|---------|

| className   | `string`                                     | Additional CSS classes.    | -       |
| ...props    | `ComponentProps<typeof ScrollAreaPrimitive.Root>` | Radix UI Root props.      | -       |

## ScrollArea.Viewport

| Prop        | Type                                         | Description                        | Default |
|-------------|----------------------------------------------|------------------------------------|---------|
| children    | `ReactNode`                                  | Content displayed in the scroll area. | -       |
| className   | `string`                                     | Additional CSS classes.            | -       |
| ...props    | `ComponentProps<typeof ScrollAreaPrimitive.Viewport>` | Radix UI Viewport props.   | -       |

## ScrollArea.ScrollBar

| Prop        | Type                                         | Description                | Default |
|-------------|----------------------------------------------|----------------------------|---------|
| orientation | `"vertical" | "horizontal"`                  | Scrollbar orientation.     | "vertical" |
| className   | `string`                                     | Additional CSS classes.    | -       |
| ...props    | `ComponentProps<typeof ScrollAreaPrimitive.ScrollAreaScrollbar>` | Radix UI Scrollbar props. | -       |
