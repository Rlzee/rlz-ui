## TextShimmer (Main)

Applies an animated shimmer gradient to text. Uses Framer Motion for a continuous animation loop and adjusts spread based on the text length.

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| children | `string` | Text content to apply the shimmer effect to. | `-` |
| as | `React.ElementType` | Element type to render (e.g., 'p', 'span', 'h1'). | `'p'` |
| className | `string` | Additional classes applied to the wrapper element. | `''` |
| duration | `number` | Animation duration in seconds for the shimmer loop. | `2` |
| spread | `number` | Multiplier controlling the shimmer spread relative to the text length. | `2` |
