# Installation
```bash
npm: npx rlz@latest add label
pnpm: pnpm dlx rlz@latest add label
yarn: yarn rlz@latest add label
bun: bunx --bun rlz@latest add label
```

# Usage
```tsx
import { Label } from "@ui/components/label"
```

```tsx
<Label htmlFor="email">Email</Label>
```

# Props
<Separator orientation="horizontal" />
<ComponentProps name="label" />

<MarkdownFallback>

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| htmlFor | `string` | Associates the label with an input element by ID. | `undefined` |
| asChild | `boolean` | Renders the label as another component via Slot. | `false` |
| className | `string` | Additional CSS classes to apply to the label element. | Predefined styles |

</MarkdownFallback>
