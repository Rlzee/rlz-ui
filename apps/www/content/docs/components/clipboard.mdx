## Clipboard

The `Clipboard` component renders a small button that copies the provided `text` to the system clipboard. It exposes a few configuration options:

- `text` — The string to copy when activated.
- `size` — Icon size: `sm | md | lg` (maps to CSS size classes).
- `position` — `relative` or `absolute` helper for overlay placement.
- `showTooltip` / `tooltipContent` — Controls the helper tooltip and the text shown while in the copied state.
- `successDuration` — How long (ms) the copied state persists before reverting.
- `onCopy` — Callback invoked after a successful copy.
- `children` — Provide custom trigger content to replace the default icons.

Use the demo preview above to interact with the component.
