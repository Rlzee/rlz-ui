# Installation

```bash
npm: npx rlz@latest add select

bun: bunx --bun rlz@latest add select
```

# Usage
```tsx
import { Select } from "@ui/components/select";
```

```tsx
export default function Example() {
  return (
    <Select>
      <Select.Trigger placeholder="Select a value" />
      <Select.Content>{/* groups / items */}</Select.Content>
    </Select>
  );
}
```

## Select

The `Select` component is a composed control exposing several slots:

- `main` — Root container.
- `Trigger` — The clickable/selectable trigger (supports `placeholder`).
- `Content` — Popup/overlay holding groups and items.
- `Group` — Logical grouping for items with an optional `Label`.
- `Item` — Selectable option (provides `value`).
- `Separator` — Visual divider between groups.

Use the demo preview above to explore the component and its slots.
