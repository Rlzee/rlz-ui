# Installation
```bash
npm: npx rlz@latest add textarea
pnpm: pnpm dlx rlz@latest add textarea
yarn: yarn rlz@latest add textarea
bun: bunx --bun rlz@latest add textarea
```

# Usage
```tsx
import { Textarea } from "@ui/components/textarea"
```

```tsx
<Textarea placeholder="Enter your message..." />
```
