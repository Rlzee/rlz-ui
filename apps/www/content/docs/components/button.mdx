# Installation
```bash
npm: npx rlz@latest add button
pnpm: pnpm dlx rlz@latest add button
yarn: yarn rlz@latest add button
bun: bunx --bun rlz@latest add button
```

# Usage
```tsx
import { Button } from "@ui/components/button"
```

```tsx
<Button variant="primary">Button</Button>
```

# Props
<Separator orientation="horizontal" />
<ComponentProps name="button" />

<MarkdownFallback>

| Prop | Type | Description | Default |
|------|------|-------------|---------|
| variant | `"primary"` \| `"secondary"` \| `"outline"` \| `"ghost"` \| `"destructive"` \| `"link"` | Sets the button style. | `"primary"` |
| size | `"default"` \| `"sm"` \| `"lg"` \| `"icon"` | Sets the button size. | `"default"` |
| asChild | `boolean` | Renders the button as another component via Slot. | `false` |
| `className` | `string` | Predefined styles | Additional CSS classes to apply to the button element |
| ...props | All native `<button>` element props | Standard HTML props for `<button>`. |  |
