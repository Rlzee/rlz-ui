# Installation
```bash
npm: npx rlz@latest add separator-border
pnpm: pnpm dlx rlz@latest add separator-border
yarn: yarn rlz@latest add separator-border
bun: bunx --bun rlz@latest add separator-border
```

# Usage
```tsx
import { SeparatorBorder } from "@ui/components/separator-border"
```

```tsx
<div className="space-y-4">
  <p>Content above</p>
  <SeparatorBorder orientation="horizontal" />
  <p>Content below</p>
</div>
```
