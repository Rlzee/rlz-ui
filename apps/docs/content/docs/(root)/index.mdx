rlz-ui isn't just about installing components—it's about building an interface that matches your product without creating a design system from scratch.

It provides production-ready components that work out of the box while giving you the freedom to customize every part of your interface. Start with the default experience or shape it into something that feels entirely your own.

## Why rlz-ui?

Every project starts somewhere.

Some teams simply need reliable components. Others want a distinctive visual identity. Eventually, many end up building their own design system.

rlz-ui is designed to support that entire journey.

You can install components and start building immediately. As your project evolves, you can customize colors, typography, animations, icon libraries and UI presets, allowing the same foundation to adapt to completely different products.

The goal isn't to replace design systems.

It's to make creating one dramatically easier.

## Philosophy

Most UI libraries stop at components.

rlz-ui starts there.

The long-term vision is a platform where every project can build, customize and share its own UI identity.

Instead of creating an entirely new component library, you can start from rlz-ui and customize every visual aspect of it. Eventually, complete presets, component packs and community contributions can be installed in just a few commands.

Whether you only change a few colors or completely transform the look and feel, you're always building on the same reliable foundation.
