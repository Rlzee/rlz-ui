## Usage

```tsx
import { Meter } from "@/components/ui/meter";
```

### Shorthand

```tsx
<Meter value={72} min={0} max={100} />
```

### Compound

```tsx
<Meter.Root value={72} min={0} max={100}>
  <Meter.Label>Label</Meter.Label>
  <Meter.Value />
  <Meter.Track>
    <Meter.Indicator />
  </Meter.Track>
</Meter.Root>
```
