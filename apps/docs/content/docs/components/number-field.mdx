## Usage

```tsx
import { NumberField } from "@/components/ui/number-field";
```

```tsx
<NumberField defaultValue={0}>
  <NumberField.ScrubArea label="Quantity" />
  <NumberField.Group>
    <NumberField.Decrement />
    <NumberField.Input />
    <NumberField.Increment />
  </NumberField.Group>
</NumberField>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/number-field/#api-reference) counterpart.
Only additions and modifications are listed here.

### Number Field Scrub Area

| Prop              | Type      | Default |
| ----------------- | --------- | ------- |
| `label`           | `string`  | -       |
| `scrubAreaCursor` | `boolean` | `true`  |

`label` is required for accessibility — it renders as a `<Label>` linked to the input.
When `scrubAreaCursor` is `false`, the custom cursor icon is not rendered.

### Number Field Scrub Area Cursor

| Prop     | Type                                                 | Default          |
| -------- | ---------------------------------------------------- | ---------------- |
| `cursor` | `React.ComponentType<React.SVGProps<SVGSVGElement>>` | `CursorGrowIcon` |

### Number Field Decrement

| Prop   | Type              | Default     |
| ------ | ----------------- | ----------- |
| `icon` | `React.ReactNode` | `<Minus />` |

### Number Field Increment

| Prop   | Type              | Default    |
| ------ | ----------------- | ---------- |
| `icon` | `React.ReactNode` | `<Plus />` |
