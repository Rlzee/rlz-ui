## Usage

```tsx
import { Popover } from "@/components/ui/popover";
```

```tsx
<Popover>
  <PopoverTrigger>Open Popover</PopoverTrigger>
  <PopoverPopup>
    <PopoverTitle>Popover Title</PopoverTitle>
    <PopoverDescription>Popover Description</PopoverDescription>
    <PopoverClose>Close</PopoverClose>
  </PopoverPopup>
</Popover>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/popover/#api-reference) counterpart.
Only additions and modifications are listed here.

### Popover Popup

Includes `Portal`, `Backdrop`, and `Positioner` internally.

| Prop              | Type                                              | Default             |
| ----------------- | ------------------------------------------------- | ------------------- |
| `backdrop`        | `boolean`                                         | `false`             |
| `backdropProps`   | `React.ComponentProps<typeof PopoverBackdrop>`    | -                   |
| `positionerProps` | `React.ComponentProps<typeof PopoverPositioner>`  | `{ sideOffset: 6 }` |

### Popover Arrow

Extends Base Component [Popup Arrow](/docs/base-components#popup-arrow).
