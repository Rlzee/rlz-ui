## Usage

```tsx
import { Button } from "@/components/ui/button";
import { Group } from "@/components/ui/group";
```

```tsx
<Group>
  <Button>Button</Button>
  <Group.Separator />
  <Button>Button</Button>
</Group>
```

## API Reference

This is a custom component, not a Base UI wrapper.

### Group (Root)

| Prop          | Type                         | Default        |
| ------------- | ---------------------------- | -------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |

### Group Separator

Adapts automatically to the parent `<Group>` orientation. Renders as a styled `<div>`.

### Group Text

Styled text or label inside a group. Supports the `render` prop for custom element rendering.
