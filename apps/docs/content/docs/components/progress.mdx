## Usage

```tsx
import { Progress } from "@/components/ui/progress";
```

### Shorthand

```tsx
<Progress label="label" value={72} min={0} max={100} />
```

### Compound

```tsx
<Progress.Root value={72} min={0} max={100}>
  <Progress.Label>Label</Progress.Label>
  <Progress.Value />
  <Progress.Track>
    <Progress.Indicator />
  </Progress.Track>
</Progress.Root>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/progress/#api-reference) counterpart.
Only additions and modifications are listed here.

### Progress

Shorthand that composes `<Progress.Root>`, `<Progress.Label>`, `<Progress.Value>`, `<Progress.Track>`, and `<Progress.Indicator>`.

| Prop         | Type      | Default |
| ------------ | --------- | ------- |
| `label`      | `string`  | -       |
| `valueLabel` | `boolean` | `false` |
