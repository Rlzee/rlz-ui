## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/input/#api-reference) counterpart.
Only additions and modifications are listed here.

### Input

| Prop          | Type                          | Default     |
| ------------- | ----------------------------- | ----------- |
| `variant`     | `"primary" \| "secondary"`    | `"primary"` |
| `unstyled`    | `boolean`                     | `false`     |
| `nativeInput` | `boolean`                     | `false`     |

When `nativeInput` is `true`, renders a plain `<input>` instead of the Base UI Input — useful when embedding inside other primitives that manage their own input rendering (e.g. `<Combobox.Input>`).
