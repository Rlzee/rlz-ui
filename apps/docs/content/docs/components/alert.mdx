## Usage

```tsx

```

```tsx
<Alert>
  <Alert.Title>Notice</Alert.Title>
  <Alert.Description>
    This is a default alert used to display general information.
  </Alert.Description>
</Alert>
```

## API Reference

This is a custom component, not a Base UI wrapper.

### Alert (Root)

| Prop      | Type                                                       | Default     |
| --------- | ---------------------------------------------------------- | ----------- |
| `variant` | `"default" \| "error" \| "info" \| "success" \| "warning"` | `"default"` |

### Alert Title

Title text for the alert. Renders as a styled `h3`.

### Alert Description

Description text for the alert. Renders as a styled `p`.

### Alert Action

Container for action buttons. Automatically positions to the right on larger screens.
