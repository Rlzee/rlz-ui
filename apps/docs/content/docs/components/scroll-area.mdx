## Usage

```tsx
import { ScrollArea } from "@/components/ui/scroll-area";
```

### Shorthand

```tsx
<ScrollArea className="h-64 rounded-md border">
  <p>
    Vernacular architecture is building done outside any academic tradition, and
    without professional guidance. It is not a particular architectural movement
    or style, but rather a broad category, encompassing a wide range and variety
    of building types, with differing methods of construction, from around the
    world, both historical and extant and classical and modern. Vernacular
    architecture constitutes 95% of the world's built environment, as estimated
    in 1995 by Amos Rapoport, as measured against the small percentage of new
    buildings every year designed by architects and built by engineers.
  </p>
</ScrollArea>
```

### Compound

```tsx
<ScrollArea.Root className="h-64 rounded-md border">
  <ScrollArea.Viewport>
    <p>
      Vernacular architecture is building done outside any academic tradition,
      and without professional guidance. It is not a particular architectural
      movement or style, but rather a broad category, encompassing a wide range
      and variety of building types, with differing methods of construction,
      from around the world, both historical and extant and classical and
      modern. Vernacular architecture constitutes 95% of the world's built
      environment, as estimated in 1995 by Amos Rapoport, as measured against
      the small percentage of new buildings every year designed by architects
      and built by engineers.
    </p>
  </ScrollArea.Viewport>
  <ScrollArea.ScrollBar orientation="vertical" />
  <ScrollArea.ScrollBar orientation="horizontal" />
  <ScrollArea.ScrollAreaCorner />
</ScrollArea.Root>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/scroll-area/#api-reference) counterpart.
Only additions and modifications are listed here.

### Scroll Area

Shorthand that composes `<ScrollArea.Root>`, `<ScrollArea.Viewport>`, `<ScrollArea.Scrollbar>`, and `<ScrollArea.Corner>`.

| Prop              | Type      | Default |
| ----------------- | --------- | ------- |
| `scrollFade`      | `boolean` | `false` |
| `scrollbarGutter` | `boolean` | `false` |

### Scroll Area Viewport

| Prop              | Type      | Default |
| ----------------- | --------- | ------- |
| `scrollFade`      | `boolean` | `true`  |
| `scrollbarGutter` | `boolean` | `true`  |

`scrollFade` adds a fade mask at the edges to indicate overflowing content.
`scrollbarGutter` adds padding to prevent content from being hidden behind the scrollbar.
