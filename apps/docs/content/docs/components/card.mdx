## Usage

```tsx
import { Card } from "@/components/ui/card";
```

```tsx
<Card>
  <Card.Header>
    <Card.Title>Title</Card.Title>
    <Card.Description>This is a description</Card.Description>
  </Card.Header>
  <Card.Body>Content</Card.Body>
  <Card.Footer>This is a footer</Card.Footer>
</Card>
```

With Frame

```tsx
import { CardFrame, Card } from "@/components/ui/card";
```

```tsx
<CardFrame>
  <CardFrame.Header>
    <Card.Title>Title Frame</Card.Title>
    <Card.Description>This is a description Frame</Card.Description>
  </CardFrame.Header>
  <Card>
    <Card.Header>
      <Card.Title>Title</Card.Title>
      <Card.Description>This is a description</Card.Description>
    </Card.Header>
    <Card.Body>Content</Card.Body>
    <Card.Footer>This is a footer</Card.Footer>
  </Card>
  <CardFrame.Footer>This is a footer Frame</CardFrame.Footer>
</CardFrame>
```

## API Reference

This is a custom component, not a Base UI wrapper.

### Card Frame

Root container for the framed `<Card>` layout.

| Prop      | Type                       | Default     |
| --------- | -------------------------- | ----------- |
| `variant` | `"primary" \| "secondary"` | `"primary"` |
| `render`  | `React.ReactElement`       | -           |

### Card Frame Header

Header section for the card frame. Supports the `render` prop.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |

### Card (Root)

Container for grouping related content with a `<Card.Header>`, `<Card.Body>`, and `<Card.Footer>`.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |

### Card Header

Header section for the card. Supports the `render` prop.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |

### Card Title

Title text for the `<Card.Header>`. Renders as a styled `<h3>`.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |

### Card Description

Description text for the `<Card.Header>`. Renders as a styled `<p>`.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |

### Card Body

Body section for the card. Supports the `render` prop.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |

### Card Footer

Footer section for the card. Supports the `render` prop.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |

### Card Frame Footer

Footer section for the frame.

| Prop     | Type                 | Default |
| -------- | -------------------- | ------- |
| `render` | `React.ReactElement` | -       |
