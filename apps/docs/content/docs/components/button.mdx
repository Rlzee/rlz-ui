```

```

## API Reference

This is a custom component, using Base UI's `useRender` hook, not a direct Base UI wrapper.

### Button

| Prop      | Type                                                                                                    | Default     |
| --------- | ------------------------------------------------------------------------------------------------------- | ----------- |
| `variant` | `"primary" \| "secondary" \| "outline" \| "ghost" \| "link" \| "destructive"`                           | `"primary"` |
| `size`    | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "icon-xs" \| "icon-sm" \| "icon-md" \| "icon-lg" \| "icon-xl"` | `"md"`      |
| `render`  | `React.ReactElement`                                                                                    | -           |
