## Usage

```tsx
import { Drawer } from "@/components/ui/drawer";
```

```tsx
<Drawer>
  <Drawer.Trigger>Open</Drawer.Trigger>
  <Drawer.Popup>
    <Drawer.Header>
      <Drawer.Title>Title</Drawer.Title>
      <Drawer.Description>Description</Drawer.Description>
    </Drawer.Header>
    <Drawer.Body>Content</Drawer.Body>
    <Drawer.Footer>
      <Drawer.Close>Cancel</Drawer.Close>
    </Drawer.Footer>
  </Drawer.Popup>
</Drawer>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/drawer/#api-reference) counterpart.
Only additions and modifications are listed here.

### Drawer

| Prop      | Type                                                 | Default     |
| --------- | ---------------------------------------------------- | ----------- |
| `variant` | `"default" \| "bare" \| "bare-bottom" \| "bare-top"` | `"default"` |
| `layout`  | `"full" \| "inset"`                                  | `"full"`    |
| `indent`  | `boolean`                                            | `false`     |

`variant` controls the visual style of `<Drawer.Header>` and `<Drawer.Footer>`:

- `"default"` ŌĆö header has a bottom border, footer has a top border
- `"bare"` ŌĆö no borders on either
- `"bare-bottom"` ŌĆö footer has a top border, header has none
- `"bare-top"` ŌĆö header has a bottom border, footer has none

`layout` controls the drawer's positioning and shape:

- `"full"` ŌĆö the drawer spans edge to edge
- `"inset"` ŌĆö the drawer has margins and rounded corners

`indent` enables the indent animation on the content behind the drawer when opening.

### Drawer Popup

Includes `Portal`, `Viewport`, and `Backdrop` internally.

| Prop              | Type                                          | Default |
| ----------------- | --------------------------------------------- | ------- |
| `backdrop`        | `boolean`                                     | `true`  |
| `backdropBlur`    | `boolean`                                     | -       |
| `showCloseButton` | `boolean`                                     | `true`  |
| `portalProps`     | `React.ComponentProps<typeof DrawerPortal>`   | -       |
| `viewportProps`   | `React.ComponentProps<typeof DrawerViewport>` | -       |

### Drawer Header

Styled header rendered as a `<div>`. Appearance is controlled by `variant` and `layout` on `<Drawer>`.
Supports the `render` prop for custom element rendering.

### Drawer Body

Scrollable content area wrapping children in a `<ScrollArea>`.

| Prop              | Type      | Default |
| ----------------- | --------- | ------- |
| `scrollFade`      | `boolean` | `true`  |
| `scrollClassName` | `string`  | -       |

Supports the `render` prop for custom element rendering.

### Drawer Footer

Styled footer rendered as a `<div>`. Appearance is controlled by `variant` and `layout` on `<Drawer>`.
Supports the `render` prop for custom element rendering.

### Drawer Backdrop

Extends Base Component [Backdrop](/docs/base-components#backdrop).

See the [Base UI](https://base-ui.com/react/components/drawer/#api-reference) documentation for more information.
