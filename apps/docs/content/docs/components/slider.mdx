## Usage

```tsx
import { Slider } from "@/components/ui/slider";
```

### Shorthand

```tsx
<Slider defaultValue={50} />;
```

### Compound

```tsx
<Slider.Root defaultValue={20}>
  <Slider.Value />
  <Slider.Control>
    <Slider.Track>
      <Slider.Indicator />
      <Slider.Thumb />
    </Slider.Track>
  </Slider.Control>
</Slider.Root>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/slider/#api-reference) counterpart.
Only additions and modifications are listed here.

### Slider

Shorthand that composes `<Slider.Root>`, `<Slider.Value>`, `<Slider.Control>`, `<Slider.Track>`, `<Slider.Indicator>`, and `<Slider.Thumb>`.

| Prop        | Type      | Default |
| ----------- | --------- | ------- |
| `showValue` | `boolean` | `false` |
