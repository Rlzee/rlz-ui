## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/toggle/#api-reference) counterpart.
Only additions and modifications are listed here.

### Toggle

| Prop      | Type                                                                                                    | Default     |
| --------- | ------------------------------------------------------------------------------------------------------- | ----------- |
| `variant` | `"default" \| "outline"`                                                                                | `"default"` |
| `size`    | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "icon-xs" \| "icon-sm" \| "icon-md" \| "icon-lg" \| "icon-xl"` | `"md"`      |

### Toggle Group

`variant` and `size` set on `<Toggle.Group>` are inherited by all child `<Toggle>` components.

| Prop          | Type                         | Default        |
| ------------- | ---------------------------- | -------------- |
| `variant`     | `"default" \| "outline"`     | `"default"`    |
| `size`        | `"xs" \| "sm" \| ...`        | `"md"`         |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |

### Toggle Group Separator

Visual divider between toggles inside a `<Toggle.Group>`. Adapts automatically to the group orientation.
