## Usage

```tsx
import { Dialog } from "@/components/ui/dialog";
```

```tsx
<Dialog>
  <Dialog.Trigger>Open Dialog</Dialog.Trigger>
  <Dialog.Popup>
    <Dialog.Header>
      <Dialog.Title>Dialog Title</Dialog.Title>
      <Dialog.Description>Dialog Description</Dialog.Description>
    </Dialog.Header>
    <Dialog.Body>Content</Dialog.Body>
    <Dialog.Footer>
      <Dialog.Close>Close</Dialog.Close>
    </Dialog.Footer>
  </Dialog.Popup>
</Dialog>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/dialog/#api-reference) counterpart.
Only additions and modifications are listed here.

### Dialog

| Prop      | Type                                                 | Default     |
| --------- | ---------------------------------------------------- | ----------- |
| `variant` | `"default" \| "bare" \| "bare-bottom" \| "bare-top"` | `"default"` |

`variant` controls the visual style of `<Dialog.Header>` and `<Dialog.Footer>`:

- `"default"` â€” header has a bottom border, footer has a top border
- `"bare"` â€” no borders on either
- `"bare-bottom"` â€” footer has a top border, header has none
- `"bare-top"` â€” header has a bottom border, footer has none

### Dialog Popup

Includes `Portal`, `Viewport`, and `Backdrop` internally.

| Prop              | Type      | Default |
| ----------------- | --------- | ------- |
| `backdrop`        | `boolean` | `true`  |
| `backdropBlur`    | `boolean` | -       |
| `showCloseButton` | `boolean` | `true`  |

### Dialog Header

Styled header rendered as a `<div>`. Appearance is controlled by `variant` on `<Dialog>`.
Supports the `render` prop for custom element rendering.

### Dialog Body

Scrollable content area wrapping children in a `<ScrollArea>`.

| Prop              | Type      | Default |
| ----------------- | --------- | ------- |
| `scrollFade`      | `boolean` | `true`  |
| `scrollClassName` | `string`  | -       |

Supports the `render` prop for custom element rendering.

### Dialog Footer

Styled footer rendered as a `<div>`. Appearance is controlled by `variant` on `<Dialog>`.
Supports the `render` prop for custom element rendering.

### Dialog Backdrop

Extends Base Component [Backdrop](/docs/base-components#backdrop).

See the [Base UI](https://base-ui.com/react/components/combobox/#api-reference) documentation for more information.
