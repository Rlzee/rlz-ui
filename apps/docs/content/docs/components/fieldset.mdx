## Usage

```tsx
import { Fieldset } from "@/component/ui/fieldset";
```

```tsx
<Fieldset>
  <Fieldset.Header>
    <Fieldset.Legend>Legend</Fieldset>
    <Fieldset.Description>Description</Fieldset.Description>
  </Fieldset.Header>
</Fieldset>
```

## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/fieldset/#api-reference) counterpart.
Only additions and modifications are listed here.

### Fieldset Header

Wrapper for `<Fieldset.Legend>` and `<Fieldset.Description>`. Renders as a styled `<div>`.

### Fieldset Description

Renders as a styled `<p>`.
