## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/select/#api-reference) counterpart.
Only additions and modifications are listed here.

### Select Trigger

| Prop   | Type                                   | Default |
| ------ | -------------------------------------- | ------- |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`  |

### Select Icon

| Prop   | Type                                                 | Default       |
| ------ | ---------------------------------------------------- | ------------- |
| `icon` | `React.ComponentType<React.SVGProps<SVGSVGElement>>` | `ChevronDown` |

### Select Popup

Includes `Portal`, `Backdrop`, and `Positioner` internally.

| Prop              | Type                                                           | Default |
| ----------------- | -------------------------------------------------------------- | ------- |
| `backdrop`        | `boolean`                                                      | `false` |
| `backdropProps`   | `React.ComponentProps<typeof SelectBackdrop>`                  | -       |
| `positionerProps` | `React.ComponentProps<typeof SelectPositioner>`                | -       |
| `scrollUp`        | `boolean`                                                      | `true`  |
| `scrollDown`      | `boolean`                                                      | `true`  |
| `scrollUpProps`   | `React.ComponentProps<typeof SelectPrimitive.ScrollUpArrow>`   | -       |
| `scrollDownProps` | `React.ComponentProps<typeof SelectPrimitive.ScrollDownArrow>` | -       |
