## Usage

```tsx

```

```tsx
<Empty>
  <Empty.Header>
    <Empty.Media>
      <Icon />
    </Empty.Media>
    <Empty.Title>No data</Empty.Title>
    <Empty.Description>No data found</Empty.Description>
  </Empty.Header>
  <Empty.Actions>
    <Button>Add data</Button>
  </Empty.Actions>
</Empty>
```

## API Reference

This is a custom component, not a Base UI wrapper.

### Empty (Root)

Container for all sub-components. Renders as a styled `<div>`.

### Empty Media

| Prop      | Type                       | Default     |
| --------- | -------------------------- | ----------- |
| `variant` | `"default" \| "secondary"` | `"default"` |
| `size`    | `"sm" \| "md" \| "lg"`     | `"md"`      |

`size` only applies when `variant="default"`.

### Empty Header

Wrapper for `<Empty.Title>` and `<Empty.Description>`. Renders as a styled `<div>`.

### Empty Title

Renders as a styled `<h3>`.

### Empty Description

Renders as a styled `<p>`.

### Empty Actions

Container for action buttons. Renders as a styled `<div>`.
