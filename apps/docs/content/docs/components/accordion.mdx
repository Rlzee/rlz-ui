## Usage

```tsx
import { Accordion } from "@/components/ui/accordion";
```

```tsx
<Accordion multiple>
  <Accordion.Item>
    <Accordion.Trigger>Accordion Item 1</Accordion.Trigger>
    <Accordion.Panel>
      Lorem ipsum dolor sit amet consectetur adipisicing elit. Iste, minima.
      Mollitia expedita, doloribus dolores quae officia esse asperiores odio
      quibusdam hic placeat ducimus iste. Obcaecati ipsum omnis minima adipisci
      tempora?
    </Accordion.Panel>
  </Accordion.Item>
  <Accordion.Item>
    <Accordion.Trigger>Accordion Item 2</Accordion.Trigger>
    <Accordion.Panel>
      Lorem ipsum dolor sit amet consectetur, adipisicing elit. Nemo placeat
      quibusdam porro voluptates, suscipit neque quisquam fugit dolore quasi.
      Explicabo ipsam doloribus adipisci numquam corporis expedita quas labore
      officia quisquam.
    </Accordion.Panel>
  </Accordion.Item>
</Accordion>
```
