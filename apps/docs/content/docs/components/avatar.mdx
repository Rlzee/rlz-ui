## Usage

```tsx
import { Avatar } from "@/components/ui/avatar";
```

```tsx
<Avatar src="/example.png" alt="Example" fallback="EX" />
```

Or

```tsx
<Avatar>
  <Avatar.Image src="/example.png" alt="Example" />
  <Avatar.Fallback>EX</Avatar.Fallback>
</Avatar>
```

## API Reference

### Avatar

| Prop       | Type                                   | Default    |
| ---------- | -------------------------------------- | ---------- |
| `src`      | `string`                               | -          |
| `alt`      | `string`                               | `"Avatar"` |
| `fallback` | `string`                               | `"A"`      |
| `size`     | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`     |
| `loading`  | `"eager" \| "lazy"`                    | `"lazy"`   |

### Avatar (Root)

| Prop   | Type                                   | Default |
| ------ | -------------------------------------- | ------- |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`  |

### Avatar Stack

| Prop    | Type                                   | Default |
| ------- | -------------------------------------- | ------- |
| `space` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`  |

See the [Base UI](https://base-ui.com/react/components/avatar/#api-reference) documentation for more information.
