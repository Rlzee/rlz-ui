## Usage

```tsx
import { Avatar } from "@/components/ui/avatar";
```

```tsx
<Avatar src="/example.png" alt="Example" fallback="EX" />
```

Or

```tsx
<Avatar>
  <Avatar.Image src="/example.png" alt="Example" />
  <Avatar.Fallback>EX</Avatar.Fallback>
</Avatar>
```
