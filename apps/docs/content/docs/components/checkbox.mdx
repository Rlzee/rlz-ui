## Usage

```tsx
import { Checkbox } from "@/components/ui/checkbox";
```

```tsx
<Checkbox />
```

Or

```tsx
<Checkbox.Root>
  <Checkbox.Indicator>
    <Checkbox.Icon />
  </Checkbox.Indicator>
</Checkbox.Root>
```

## API Reference

### Checkbox

| Prop                | Type                                                 | Default   |
| ------------------- | ---------------------------------------------------- | --------- |
| `icon`              | `React.ComponentType<React.SVGProps<SVGSVGElement>>` | `"Check"` |
| `indeterminateIcon` | `React.ComponentType<React.SVGProps<SVGSVGElement>>` | `"Minus"` |

### Checkbox Icon

| Prop   | Type                                                 | Default   |
| ------ | ---------------------------------------------------- | --------- |
| `icon` | `React.ComponentType<React.SVGProps<SVGSVGElement>>` | `"Check"` |

See the [Base UI](https://base-ui.com/react/components/checkbox#api-reference) documentation for more information.

For checkbox group, see the [CheckboxGroup](https://base-ui.com/react/components/checkbox-group#api-reference) in Base UI documentation.
