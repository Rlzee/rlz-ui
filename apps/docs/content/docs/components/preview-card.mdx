## API Reference

Each component extends its [Base UI](https://base-ui.com/react/components/preview-card/#api-reference) counterpart.
Only additions and modifications are listed here.

### Preview Card Popup

Includes `Portal`, `Backdrop`, and `Positioner` internally.

| Prop              | Type                                                    | Default             |
| ----------------- | ------------------------------------------------------- | ------------------- |
| `backdrop`        | `boolean`                                               | `false`             |
| `backdropProps`   | `React.ComponentProps<typeof PreviewCardBackdrop>`      | -                   |
| `positionerProps` | `React.ComponentProps<typeof PreviewCardPositioner>`    | `{ sideOffset: 6 }` |

### Preview Card Arrow

Extends Base Component [Popup Arrow](/docs/base-components#popup-arrow).
